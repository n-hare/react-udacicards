# UdaciCards n-hare

## Application Install

### Run
### `yarn`
### `yarn start`

## Main Views
- Index
  - Deck List
    - lists all decks
  - New Deck
    - form to create new deck

- Single Deck
  - display all deck details
  - Add Card
    - form to create new card with question and answer
  - Quiz
    - view the cards within deck
    - receive total correct

## Notes
### Targets iOS
### Uses redux-persist to maintain state in AsyncStorage
